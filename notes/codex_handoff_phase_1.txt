  line==redis_line => break, regex match => row, no-match+blank => skip,
  no-match+non-blank => "Bad Match" WARN + skip + count (TD-017 divergence; Node
  would crash here).
- mapDataToSchema parity: domain_group/id_group/month/day/year never captured by
  win_10 regex => null. host_datetime/capture_datetime => null in Phase 1 (Phase 2).
- CRLF (TD-016): files are CRLF on the dev host (verified: 2187/2187 lines). Each
  line has its trailing \r stripped before compare/parse.
